# TO-DO List
Ten projekt jest tzw. CRUDem, czyli aplikacją, która potrafii:
* Tworzyć  (C - Create)
* Czytać   (R - Read)
* Zmieniać (U - Update)
* Usuwać   (D - Delete)
  dane w bazie danych.

Aplikacja pozwala na zarządzanie TO-DO Listą. Każdy tasks na tej liście zawiera: nazwę, opis i długość czasu od powstania pliku.
